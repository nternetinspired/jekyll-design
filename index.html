This is a rapid prototyping and comping tool built with Jekyll by Seth Warburton to enable (his own) rapid in-browser design, from wireframes and mockups right through to hi-fidelity html.

Nice.

Sassy Styles

It also includes neato design tools like my Sass interpretation of Samanth Warren's Style Tiles and some other patterns that allow you to quickly create color pallettes using Sass' built-in colour functions. It also includes a customised version of my own Sass framework, oneweb-sass. Of course, it's super-simple to rip out my Sass and replace it with your own. Whatever floats your boat.

For more information on Style tiles I recommend reading this excellent ALA article; Style Tiles and How They Work by their creator.

You can find the source code for it at: Github.
