This is a rapid prototyping and comping tool built with Jekyll by Seth Warburton to enable (his own) rapid in-browser design, from wireframes and hi-fidelity mockups right through to production html and css. Nice.

Mmmmm Sassy

It includes neato design tools like my Sass interpretation of Samantha Warren's Style Tiles and some other patterns that allow you to quickly create color pallettes using Sass' built-in colour functions. It also includes a customised version of my own Sass framework, oneweb-sass. Of course, it's super-simple to rip out my Sass and replace it with your own. Whatever floats your boat.

Items of note:

  • Fully awesomised Jekyll.
  • Github pages friendly. Clone > tweak > push > view.
  • Minified html output (optional).
  • oneweb sass
  • Style tiles
  • Auto-swatches

You can find the source code for it at: Github. To run locally you just need to fire up terminal and enter: 

jekyll serve --watch --baseurl
